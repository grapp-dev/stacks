`Stack` distributes white space evenly between components (vertically or horizontally if you set the `horizontal` prop to `true`).

:::note

`Stack` component accepts [`View`](https://reactnative.dev/docs/view) props.

:::

## Examples

Set spacing between elements stacked vertically.

```jsx live
function() {
  return (
    <App>
      <Stack space={1}>
        <Placeholder />
        <Placeholder />
        <Placeholder />
      </Stack>
    </App>
  )
}
```

Multiple `Stack` components can be nested.

```jsx live
function() {
  return (
    <App>
      <Stack space={10}>
        <Stack space={2}>
          <Placeholder />
          <Placeholder />
        </Stack>
        <Stack space={2}>
          <Placeholder />
          <Placeholder />
          <Placeholder />
        </Stack>
      </Stack>
    </App>
  )
}
```

Set horizontal alignment.

```jsx live
function() {
  return (
    <App>
      <Stack space={1} align="center">
        <Placeholder width={250} />
        <Placeholder width={180} />
        <Placeholder width={200} />
      </Stack>
    </App>
  )
}
```

Stack elements horizontally.

```jsx live
function() {
  return (
    <App>
      <Stack space={1} horizontal={true} align="center">
        <Placeholder width={100} height={100} />
        <Placeholder width={100} height={140} />
        <Placeholder width={100} height={80} />
      </Stack>
    </App>
  )
}
```
